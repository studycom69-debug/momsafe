# MomSafe AI

**AI-powered maternal health companion** — track vitals, get real-time health risk scores, nutrition guidance, medication reminders, and AI-driven insights throughout your pregnancy journey.

🌐 **Live:** [momsafe.vercel.app](https://momsafe.vercel.app)

---

## Features

- **Command Center** — real-time health risk scoring and guidance
- **Vitals Tracking** — log and monitor pregnancy vitals over time
- **AI Guidance** — personalized health recommendations
- **Nutrition & Medication** — diet plans and medication reminders
- **Analytics** — visualize your health trends
- **Alerts** — smart notifications based on your data
- **Google Sign-In** — secure authentication via Supabase

---

## Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | React 19 + Vite 7 |
| Styling | Tailwind CSS v4 |
| Auth & Database | Supabase (PostgreSQL + Google OAuth) |
| Routing | Wouter |
| UI Components | Radix UI + shadcn/ui |
| Maps | Leaflet + React Leaflet |
| Hosting | Vercel |
